# Hello Gopher
Hello Gopher is a sample Golang package for testing to go get and call package function.

# Usage
## Install
``` sh
go get github.com/byounghoonkim/gopher
```

## Call function

``` go
package main

import "github.com/byounghoonkim/gopher"

func main() {
	gopher.Says()
}
```
